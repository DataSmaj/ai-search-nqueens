# AI Search Algorithms: N-Queens

This project explores local search algorithms for solving the N-Queens problem in Python.

## Project Overview

The N-Queens problem asks how to place N queens on an N x N chessboard so that no two queens attack each other. This project compares several search strategies and analyzes how each one performs.

## Algorithms Used

- Hill climbing
- Hill climbing with sideways moves
- Random restart hill climbing
- Simulated annealing
- Local beam search

## Tools

- Python
- Algorithm analysis
- Artificial intelligence search methods

## What I Learned

This project helped me understand how AI search algorithms behave when dealing with local minima, plateaus, randomness, and exploration. I also learned how different strategies can improve the chances of finding a valid solution.

## Files

- `n_queens.py` - N-Queens problem setup

- `pb_search_lb.py` - Local beam search implementation
